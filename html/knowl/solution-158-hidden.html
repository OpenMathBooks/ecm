\(S(k, k-2) = \binom{k}{3} + \frac{1}{2}\binom{k}{2}\binom{k-2}{2}\text{.}\) Because we have two fewer blocks than we have elements, there will always either be a block of size three and \(k-3\) blocks of size one or two blocks of size two and \(k-4\) blocks of size 1. The first case accounts for \(\binom{k}{3}\) partitions (once you have chosen the three elements to be in the large block, you are done). The second case accounts for \(\frac{1}{2}\binom{k}{2}\binom{k-2}{2}\) partitions (pick 2 elements to be together, then 2 of the remaining to be together, but don't care which order you pick these in).
