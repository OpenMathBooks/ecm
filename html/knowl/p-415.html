The problems in Activity 54 are sometimes referred to as “distribution problems”, since we are asking for the number of ways to distribute some objects to some recipients. This is a useful model for the other counting techniques we have seen as well. For example, a permutation \(P(n,k)\) gives the number of ways to distribute \(n\) distinct objects to \(k\) distinct recipients, so that each recipient gets exactly one object.
