One circle defines two regions, the inside and outside. When we draw a second circle that intersects the first, we can start at one of the intersection points and go inside the first circle, cutting its region into two pieces, and then when we leave it we cut the outside region into two pieces. This suggests the general pattern. If we have drawn \(n-1\) circles and start a new one, each time we enter a circle, we start dividing a region into two pieces. Each time we leave a circle, we also start dividing a region into two pieces. Thus if we have \(r_n\) regions with \(n\) circles, to get the number of regions, we note that in going from \(n-1\) circles to \(n\) circles, we start with \(r_{n-1}\) regions, and divide \(2(n-1)\) of them in half, so we get \(2n-2\) new regions. This gives us \(r_n=r_{n-1}+2(n-1)\text{.}\) Notice that by substitution of the formula \(r_{n-1}=r_{n-2} +2(n-2)\text{,}\) we get \(r_n=r_{n-2}+ 2(n-2) +2(n-1)\text{,}\) and would guess that \(r_n=r_{n-3}+2(n-3)+2(n-2) +2(n-1)\text{.}\) This leads to the conjecture

\begin{equation*} r_n=r_1 +2\cdot1+2\cdot2+\cdots+2\cdot(n-1)=r_1+2\sum_{i=1}^{n-1} i=2+n(n-1). \end{equation*}
