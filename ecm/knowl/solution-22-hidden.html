\(\binom{n}{k}\) is the number of subsets of \(\{ a_{1},a_{2},a_{3},\ldots,a_{n}\}\) of size \(k\text{.}\) Now a subset \(A\) of size \(k\) either contains the fixed element \(a_{i}\) or it does not. If \(A\) contains \(a_{i}\text{,}\) the remaining \(k - 1\) elements can be selected in \(\binom{n - 1}{k - 1}\) ways. If, on the other hand, \(A\) does not contain \(a_{i}\text{,}\) you can choose the \(k\) elements from the depressed set \(\left\{ a_{1},a_{2},\ldots,a_{i - 1,}a_{i + 1,\ldots,}a_{n} \right\}\) in \(\binom{n - 1}{k}\) ways. Since these two cases are mutually exclusive the theorem follows.
