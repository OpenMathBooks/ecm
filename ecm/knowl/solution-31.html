Solution 1.2.18.1.

The sum can be recharacterized using \(n^2 = \binom{n}{2}+\binom{n+1}{2}\) as follows:

\begin{align*} \amp 1^2 + \left[\binom{2}{2}+\binom{3}{2}\right]+ \left[\binom{3}{2} + \binom{4}{2}\right] + \cdots + \left[\binom{n}{2} + \binom{n+1}{2}\right] \\ = \amp \left[\binom{2}{2} + \binom{3}{2} + \binom{4}{2} + \cdots + \binom{n+1}{2}\right] + \left[\binom{2}{2} + \binom{3}{2} + \cdots + \binom{n}{2}\right] \\ = \amp \binom{n+2}{3} + \binom{n+1}{3} \end{align*}

using the Hockey Stick Theorem twice.
