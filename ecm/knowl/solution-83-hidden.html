We can describe a walk starting at \(v\) as a Dyck word: use \(a\) for each step away from \(v\) and \(t\) for each stem toward \(v\text{.}\) Since \(v\) is an endpoint of the path, no initial segment of this path can contain more \(t\)'s than \(a\)'s, so every path is a unique Dyck word. Similarly, given any Dyck word on these symbols, we have a unique path starting and ending at \(v\) (there are \(n\) \(a\)'s in our words, so the farthest away you can go from \(v\) is \(a\) steps away, and the path has length \(a\)).
