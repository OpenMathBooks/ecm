There is a \(b_0\) because it can be paired with an \(a_2\) to give the term \(a_2b_0x^4\text{.}\) Similarly there is a \(b_1\) because it can be paired with \(a_1\) for the same purpose. However there is no \(a_i\) that we can pair with \(b_0\) to get a coefficient of \(x^4\) and no \(a_i\) that we can pair with \(b_3\) to get a coefficient of \(x^4\text{.}\)
