Let \(A\) be the set of all seating arrangements for \(2n\) people around a round table. Let \(A_i\) be the set of arrangements in which couple \(i\) sits together. We are interested in \(|\overline{A_1\cup A_2\cup \cdots \cup A_n}|\text{.}\) Thus, for a set \(S\subseteq [n]\text{,}\) we need to compute \(\left|\bigcup_{i\colon i\in S} A_i\right|\text{.}\) If we let each couple described by \(S\) sit together, we will seat \(|S|\) couples and \(2n-2|S|\) individuals around the table. We can do this in \(2^{|S|}(|S| + 2n-2 |S|-1)!\) ways, because once we chose a place for a couple (i.e., two adjacent seats) there are two ways the couple can sit down. Thus as long as \(S\) is nonempty, we have the right formula for \(\left|\bigcap_{i\colon i\in S} A_i\right|\text{.}\) Notice that in the case where \(S=\emptyset\) the formula gives us \((2n-1)!\) seating arrangements, which is exactly the number of ways to seat \(2n\) people around a round table. This is the size of our set \(A\text{.}\) Therefore,

\begin{equation*} \left|\bigcap_{i\colon i\in S} A_i\right| = 2^{|S|}(2n-|S|-1)!\text{.} \end{equation*}
