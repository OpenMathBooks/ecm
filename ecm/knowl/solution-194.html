Solution 257.1.

We start by giving a function from the set of partitions of \(k\) to the set of partitions of \(k\) with (only) odd parts. Clearly such a function cannot be one to one. Then we show that when restricted to the partitions with distinct parts it is one-to-one and onto by constructing an inverse. Given a partition \(\lambda_1^{i_1}\lambda_2^{i_2}\cdots\lambda_n^{i_n}\text{,}\) write \(\lambda_i=\gamma_i2^{k_i}\text{,}\) where \(\gamma_i\) is odd. (Thus \(2^{k_i}\) is the highest power of 2 that is a factor of \(\lambda_i\text{,}\) so it is 1 if \(\lambda_i\) is odd.). It is possible that \(\gamma_i=\gamma_j\text{,}\) for example if \(\lambda_i=36\) and \(\lambda_j=18\text{,}\) then \(\gamma_i=\gamma_j=9\text{.}\) We construct a new partition \(\pi\) whose parts are the numbers \(\gamma_j\) as follows: Given an odd number \(p\text{,}\) let the multiplicity \(m(p)\) of \(p\) in \(\pi\) be \(\sum_{j: \gamma_j=m} 2^{k_j}\text{.}\) Thus \(\sum_{p: m(p)\not=0}m(p)p = k\text{.}\) Therefore, \(\pi\) is a partition of \(k\) whose parts are all odd.

Now consider a partition \(\pi\) of \(k\) whose parts are all odd. Let \(\pi=\pi_1^{r_1}\pi_2^{r_2}\cdots \pi_t^{r_t}\text{,}\) with \(\pi_i>\pi_{i+1}\text{.}\) (In terms of the multiplicity function \(m\text{,}\) \(m(\pi_i) =r_i\text{,}\) and \(\sum_{i=1}^t r_i\pi_i = k\text{.}\)) We are going to write the binary expansion of each \(r_i\) as \(r_i = \sum_{j= 0}^{\lfloor \log_2 r_i\rfloor} 2^{ja_{ij}}\text{,}\) where \(a_{ij}\) is 1 if \(2^j\) appears in the binary expansion of \(r_i\text{,}\) and 0 otherwise. All of the numbers \(\pi_i2^{ja_{ij}}\) are distinct, because a power of two times one odd number cannot equal a power of two times another odd number. The numbers \(\pi_i2^{ja_{ij}}\) add to \(k\text{,}\) so they are the parts of a partition \(\pi'\) of \(k\) into distinct parts. When we apply the function constructed in the first part of the solution to \(\pi'\text{,}\) we get \(\pi\text{,}\) so the correspondence between \(\pi\) and \(\pi'\) is a bijection.
