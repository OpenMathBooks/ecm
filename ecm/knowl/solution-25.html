Solution 1.2.13.1.

We use the “reverse and add technique”, which we will see in Subsection 1.4.2 is usually reserved for summing arithmetic sums. Surprisingly, it works here.

\(S =\) \(\binom{n}{1}^2+2\binom{n}{2}^2+3\binom{n}{3}^2 + \) \(\cdots +(n-1)\binom{n}{n-1}^2+n\binom{n}{n}^2\)
\(+ \quad S =\) \(n\binom{n}{n}^2 + \) \((n-1)\binom{n}{n-1}^2+ \cdots +\) \(2\binom{n}{2}^2+\binom{n}{1}^2\)
\(2S =\) \(n\binom{n}{0}^2 + \) \(n\binom{n}{1}^2 + n\binom{n}{2}^2+ n\binom{n}{3}^2 + \) \(\cdots + n\binom{n}{n-1}^2+n\binom{n}{n}^2\)

This uses the identity \(\binom{n}{k} = \binom{n}{n-k}\text{.}\) Now factor out an \(n\) to get

\begin{equation*} 2S = n\left[\binom{n}{0}^{2} + \binom{n}{1}^{2} + \binom{n}{2}^{2} + \ldots + \binom{n}{n}^{2}\right]\text{.} \end{equation*}

Apply the identity in Example 1.2.12 and divide by 2 to get

\begin{equation*} S = \frac{n}{2}\binom{2n}{n}\text{.} \end{equation*}
