If this new graph has an Euler circuit, then we can “cut” out the path \((v_2, v', v_1)\text{.}\) Specifically, rotate and possibly reverse the circuit so that it starts at \(v_1\) and ends with the sequence \(v_2, v', v_1\text{.}\) Then simply remove \(v', v_1\) from that sequence, to get a Euler path.
