\(\frac{n!}{\prod_{i=1}^n (i!)^{a_i}{a_i!}}\text{.}\) We can make a list in \(n!\) ways, and then break it into first \(a_1\) blocks of size 1, then \(a_2\) blocks of size 2, \(a_3\) blocks of size 3 up to \(a_n\) blocks of size \(n\text{.}\) But then we realize that we get the same partition if we permute the \(i!\) elements of a block of size \(i\) and we get the same partition if we permute the \(a_i\) blocks of size \(i\) so we apply the quotient principle.
