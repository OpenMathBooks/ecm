Since there are 8 triangles, there must be 8 faces. We can count the number of edges by taking \(8 \cdot 3 = 24\text{,}\) but this is double counting since each edge corresponds to two faces. Thus there are 12 edges. We can use Euler's formula to find that there are 6 vertices (and this shows that each vertex is the joining of 4 triangles).
