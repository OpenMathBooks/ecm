\(\binom{n}{k}\) counts the number of \(n\)-bit strings of weight \(k\text{.}\) This is because of the \(n\) bits, we must choose \(k\) to be 1's, which can be done in \(\binom{n}{k}\) ways. The number of \(n\)-bit strings of weight \(k\) is also \(\binom{n}{n-k}\text{,}\) since of the \(n\) bits, we must choose \(n-k\) to be 0's (making the remaining \(k\) 1's), which can be done in \(\binom{n}{n-k}\) ways. Thus \(\binom{n}{k} = \binom{n}{n-k}\text{.}\)
