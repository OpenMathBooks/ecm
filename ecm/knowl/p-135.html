The above is a complete proof, but really \(\binom{n}{k}\) can be thought of as defined as the size of the set \(\B^n_k\text{.}\) But then \(\binom{n}{n-k}\) is the size of the set \(\B^n_{n-k}\text{.}\) So what we are really saying here is that \(|\B^n_k| = |\B^n_{n-k}|\text{,}\) which is true because the function that switches 0's to 1's and 1's to 0's in a bit string is a bijection between these two sets.
